How It Works

Technical Process

1. Motion Detection: The system analyzes frame-by-frame motion patterns to detect movement intensity.

2. Pose Estimation: Using MediaPipe, we track key body points to understand posture and movement.

3. Activity Classification: Based on motion patterns and pose data, the system classifies the activity.

4. Real-time Feedback: Results are displayed instantly, showing the detected activity and confidence level.
